This tool adds a textbox to the interactive IPA chart at <https://www.ipachart.com/> to copy characters you click on. I made this while learning the IPA chart in a linguistics class, where we needed to transcribe speech into IPA. Listening to the interactive chart was very helpful when learning the sounds, but I needed to use a separate site to actually type the IPA characters. This script combines the two by adding each character you click on into a text box, which you can then copy and paste. 

Project on [Github](https://gist.github.com/Mdkar/6166a2aa7e7a050d001a2ab7f942a273)

**How to install and use**
* **Step 0:** install the [Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en) extension in the Chrome Web Store.
Tampermonkey is a userscript manager for Chrome. The script will run within this extension.
* **Step 1:** Install the userscript (Interactive IPA Chart Textbox) here: <https://gist.github.com/Mdkar/6166a2aa7e7a050d001a2ab7f942a273>
by clicking on the “Raw” button (a Tampermonkey page should pop up shortly and you should be able to follow the instructions to set it up)
* **Step 2:** Go to <https://www.ipachart.com/>
  * There should be a textbox with a few extra character buttons
    * Click on any interactive button on the chart and it will show up in the text box
