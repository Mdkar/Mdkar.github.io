Conestoga

Java program to make any photograph into a circular loom design using the Processing and the G4P libraries.

Direct Download (requires Java to run)

You can play with your creations, share, and embed them into a website using the PictureLoom Viewer. Instructions on how to use it along with the code are on the Github Page

If you have an embroidery machine, the program also outputs a .EXP file that allows your machine to stitch the design.
